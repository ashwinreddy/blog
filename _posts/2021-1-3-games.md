### Hopping Grass

I recently read Bernard Suits' _The Grasshopper_ (1978), a philosophical book about games. From what I gather, the book is written for philosophers,[^witt] but I enjoyed reading the first few chapters in which a working definition of "game" is outlined.[^chapters]

[^witt]: The book responds to Ludwig Wittgenstein's contention that there's nothing that can define games concretely save the vague resemblances we see between games. We will see that Suits, however, has a clear and reasonable definition.

[^chapters]: Here, I'm explaining Chapters 3.

A good place to start is with the feeling we have that games are something we do for fun, amusement, and whimsy. We might locate this amusement in the fact that we go about accomplishing goals in roundabout ways when we're playing a game. 

Consider the racetrack below. If the goal was simply to go from START to END, why waste time going around the track when you can take SHORTCUT through the grass? The answer is because the rules of the game demand it.

![](https://imgur.com/0bA8o4C.png)

We might say that games create **inefficiency** in this manner. Let's say you are playing a game $G$ with an end goal we'll denote as $\Omega(G)$. The argument is that if you only wanted to achieve $\Omega(G)$, there are more efficient ways of doing it than playing $G$. The rules only make getting to the end goal harder.[^eff]

Suits calls $\Omega(G)$ the **prelusory goal** of $G$. The word _ludo_ in Latin means "I play," and we use the prefix _pre_ to suggest that the prelusory goal can be described before knowing anything about the game rules. For example, you can describe all the configurations that result in checkmate without needing to say how the pieces move. Suits argues that no matter how complex $G$ is, you can always describe the prelusory goal as a configuration or _specific state of affairs_ without reference to the game.

[^eff]: The inefficiency is with respect to the means that are allowed in getting to the end goal. Good players are _as efficient as possible_ subject to the inefficiencies created by the rules. The runners in the track above will run as fast as they can, given that they have to run in an arc.

The reason it's hard to wrap our head around games is that we follow the rules despite the fact that all they do is make the process of getting to the end goal harder. Why would we possibly want to do that, you ask? So that we can play a game! If you aren't following the rules, you can't be said to be playing the game. We might write the following to capture the main idea here, where $A$ is a player in a game $G$
\begin{equation}
A \text{ won } G \implies A \text{ played } G \implies A \text{ followed the rules of } G
\end{equation}
You can't win a game without playing it, and if you played it, you must have followed the rules.

As it stands, this definition is too broad. To see this, we will repurpose the race track figure above as a metaphor. Here, the END will be, say, becoming the president of the local Philosophy Club. The SHORTCUT would be assassinating the current president, and the TRACK would be running for office. Clearly, we would not say that you are engaged in playing a game if you decide to run for office; the other, unsavory option is unavailable for moral and legal reasons. So we'll say that the rules must be ultimately non-binding; you don't have to play, and at any time you can stop playing.

That's pretty much it! The portable version of the main definition Suits gives is this:

> Playing a game is the voluntary attempt to overcome unnecessary obstacles.

### An Alternate Definition

This Wizards of the Coast [article](https://magic.wizards.com/en/articles/archive/making-magic/what-game-2018-06-04) also has a compelling definition for game, which shares many similarities to the Grasshopper one.

> A game is a thing with a goal (or goals), restrictions, agency, and a lack of real-world relevance.

We could identify the goal in this definition as the prelusory goal, the restrictions as the inefficiency-generating rules, agency as unstated and implicit in Suits' definition or perhaps absorbed by the phrase "attempt to overcome," and lack of real-world relevance as the ultimately-non-binding nature of the rules.

However, this articles goes one step further by explaining how the definition mutates when we ablate the components in this definition: we end up with definitions for toys, activities, events, and life itself.

### Explaining Games

In my experience, people tend to be pretty awful at explaining how a game works. Of course, "you'll pick it up as you watch" is pragmatic advice (and it can be fun to figure out the rules of a game you don't know), but perhaps these definitions can give us a principled way of explaining the game.

1. Explain the prelusory goal since, in principle, this can always be done without reference to the rules. Let the player know what they're aiming for.
2. Explain the primary means by which they can inch towards the prelusory goal.
3. Explain any secondary game mechanics or omit them for the time being if too complex.
4. Explain basic strategy of the game (i.e. why the inefficiencies are interesting).

### So What?

According to the Wizards of the Coast article, relaxing the condition on relevance gives us a definition for life, suggesting life is a generalized game or that games are mini-versions of life.

Is game-playing at the core of human experience? [^utopia] Certainly the world imposes rules (physical, societal, etc.) that limit us from getting what we want, but we decide to overcome those obstacles anyway. 

[^utopia]: Suits thinks that in a utopia, people would only play games. 

Suppose games are at the core of life. Since all games are fundamentally similar in structure and form, if we learn how to play one game well, we might then be in a better position to pick up other games as well. If we repeat this process of learning different games, we could become excellent at the game of learning-how-to-play-games. Two general strategies then emerge:

* Either learn how to play one game so well that you outclass competitors, or
* Learn the ability to pick up games quickly and find a niche for yourself

This philosophy of games might thus be connected to the existential idea that people get to set up their own goals. We'll leave the work of figuring out the interplay between goal-setting and game-playing for another day.
